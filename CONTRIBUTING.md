# Contributing Guide

Thank you for your interest in contributing! 🎉
We welcome contributions that improve this project.

---

## 🚀 Getting Started

1. Fork the repository
2. Clone your fork:

   ```bash
   git clone https://github.com/your-username/your-repo.git
   ```
3. Navigate to the project folder:

   ```bash
   cd your-repo
   ```
4. Create a new branch:

   ```bash
   git checkout -b feature/your-feature-name
   ```

---

## 🧠 Project Overview

This project is an AI-powered resume analyzer that:

* Evaluates resumes using LLMs
* Generates ATS scores
* Provides structured feedback in JSON format

---

## 🛠️ How to Contribute

You can contribute in several ways:

### 🐛 Bug Fixes

* Fix issues listed in the Issues tab
* Improve error handling

### ✨ New Features

* Enhance AI prompts
* Add UI components
* Improve API performance

### 📚 Documentation

* Improve README
* Add examples
* Fix typos or unclear explanations

---

## 🧪 Code Guidelines

* Write clean, readable code
* Follow PEP8 (for Python)
* Add comments where necessary
* Keep functions modular and reusable

---

## 🔍 Commit Message Guidelines

Use clear and descriptive commit messages:

Examples:

* `feat: add ATS score breakdown`
* `fix: handle empty resume input`
* `docs: update README with API usage`

---

## ✅ Pull Request Process

1. Ensure your code works correctly
2. Test your changes
3. Update documentation if needed
4. Submit a Pull Request with:

   * Clear description
   * Screenshots (if UI changes)
   * Linked issue (if applicable)
